# BrinkMais Infláveis 🎈

Landing Page otimizada e responsiva para a **BrinkMais**, empresa especializada em aluguel de brinquedos infláveis em Maceió, Alagoas. O projeto foca em alta performance, usabilidade e conversão direta via WhatsApp, apresentando o catálogo de atrações e reforçando a credibilidade da marca.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as ferramentas mais modernas do ecossistema front-end:

- **[React 19](https://react.dev/)** + **[TypeScript](https://www.typescriptlang.org/)**: Motor e tipagens do projeto.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build super rápida.
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Estilização baseada em utilitários e sistema de design.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para criar as animações animadas e lúdicas.
- **[Lucide React](https://lucide.dev/)**: Biblioteca de ícones moderna e leve.

### 2. Instalação

Clone ou baixe esse repositório e abra o terminal na pasta raiz do projeto (`brinkmais`), em seguida, instale as dependências:

```bash
npm install
```

### 3. Rodando o servidor de desenvolvimento

Inicie a aplicação localmente rodando o comando:

```bash
npm run dev
```

Pronto! Acesse o endereço exibido no terminal (geralmente será `http://localhost:3000` ou `http://localhost:5173`) no seu navegador de preferência.

## 📁 Estrutura de Pastas

A arquitetura do projeto (após refatoração) é simplificada para facilitar a manutenção de componentes e dados:

\`\`\`
/src
 ├── /components     # Peças estruturais e visuais da página (Hero, Navbar, Catalog, etc.)
 ├── /data           # Informações estáticas (ex: lista de brinquedos)
 ├── /utils          # Funções e regras de negócio reusáveis (ex: gerador de link do WhatsApp)
 ├── App.tsx         # Arquivo central organizador onde as seções estão montadas
 └── index.css       # Configurações globais e tokens de cor (variáveis Tailwind)
\`\`\`

## 💎 Design System & Cores
O layout segue um código de cores cartunescas e vibrantes para transparecer alegria e diversão:

- **Primary:** \`#FF3366\` (Rosa Vibrante)
- **Secondary:** \`#00C4FF\` (Azul Claro/Céu)
- **Accent:** \`#FFCC00\` (Amarelo Ouro)
- **Background:** \`#FAF7FA\` (Tom gelo quase branco)
- **Typography:** Fonte principal **Fredoka** para títulos, combinada com **Nunito** para conteúdos textuais.
